Problem Statement

Almost all of the 3D virtual reality applications currenlty utilize controllers as the primary input method. I aimed to innovate by designing and dveloping a hands-first VR creation tool that incorporates custom hand interactions.

The project's goal is to create an immersive, hands-first authoring tool by leveraging the advanced features of the Meta Interaction SDK.

Ideation & Prototyping

A significant issue with current virtual reality applications is that they often feature overly cluttered user interfaces, which predominantly rely on uninspired 2D designs.

I designed the UI controls to be intuitive and easy to use.

I started exploring finger menus which would appear on hover. The dominant hand would host the manipulation and creation actions. The non-dominant hand contains secondary functions like scale, undo/redo, and duplicate.
